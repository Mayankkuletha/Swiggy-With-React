# 🍔 Swiggy Clone (React)

A fully functional Swiggy-inspired food ordering web app built using React.  
This project demonstrates modern frontend development concepts like component-based architecture, API handling, routing, and state management.

---

## 🚀 Features

- 🔍 Search Restaurants
- 📍 Dynamic Restaurant Listing (API based)
- 🍽️ Restaurant Menu Page
- 🛒 Add to Cart functionality
- 🔄 Shimmer UI (Loading effect)
- ❌ Error handling (CORS / API issues)
- 🔗 Routing using React Router
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Parcel (Bundler)
- Swiggy Public API (for data)
- Git & GitHub
